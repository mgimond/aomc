This folder houses SPF files from different sources. They are formatted for the aomc model. 

When copying an SPF file to your project folder, be sure to rename the file `spf.inp` since the input file names are hard coded in aomc.

+  `spf.inp.petzold_3_SD_harbor_stations`: Petzold's average for the three San Diego Harbor.
+  `spf.inp.mobley`: Mobley et al.'s SPF used in their 1993 paper.
+  `spf.inp.Rayleigh`: Rayleigh scattering phase function.
