# Comparison Data

This directory contains model output files and other data used for comparing the AOMC model with other models or published results.

### mobley_problems

Folder contains digitized data from Mobley et al.'s paper and R scripts that generate the comparison tables and plots.

Mobley, C. D., Gentili, B., Gordon, H. R., Jin, Z., Kattawar, G. W., Morel, A., & Reinersman, P. (1993). Comparison of numerical models for computing the underwater light field. *Applied Optics*, *32*(36), 7484-7504. doi:10.1364/AO.32.007484

###  mobley1

Input and output data from problem 1 of Mobley paper.

###  mobley2

Input and output data from problem 2 of Mobley paper.

###  mobley6

Input and output data from problem 2 of Mobley paper.

### simple_relationship

Input and output data for the estimation of the dimensionless parameter r_p.
